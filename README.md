# Πολλαπλασιασμός Πίνακα με Διάνυσμα με χρήση Νημάτων (Java)

Υλοποίηση του γινομένου πίνακα–διανύσματος (A * v) με χρήση πολλαπλών νημάτων σε Java.

Η εργασία εστιάζει στην παράλληλη επεξεργασία, αξιοποιώντας k νήματα (όπου k είναι δύναμη του 2) για τον υπολογισμό του αποτελέσματος με αποδοτικό τρόπο.

## 📚 Πληροφορίες
- Μάθημα: ΠΛΗ 47
- Υποεργασία: Πολλαπλασιασμός Πίνακα με Διάνυσμα
- Γλώσσα: Java

## 📝 Περιγραφή

Δίνεται πίνακας A διαστάσεων n × m και διάνυσμα v διαστάσεων m × 1, όπου:
- το n είναι δύναμη του 2
- n > k
- το k (πλήθος νημάτων) είναι επίσης δύναμη του 2

Το πρόγραμμα:

- Δημιουργεί τυχαίο πίνακα A και διάνυσμα v με τιμές στο [0,10]
- Υπολογίζει το γινόμενο A * v χρησιμοποιώντας k νήματα
- Κάθε νήμα αναλαμβάνει υποσύνολο γραμμών του πίνακα
- Συλλέγει τα αποτελέσματα σε τελικό διάνυσμα
- Μετρά τον χρόνο εκτέλεσης για διαφορετικό αριθμό νημάτων

## 🛠️ Τεχνολογίες
- Java
- Java Threads
- System.nanoTime() για μετρήσεις χρόνου
